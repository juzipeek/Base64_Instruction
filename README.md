# Base64 Instruction
##Base64说明

Base64是一种基于64个可打印字符来表示二进制数据的表示方法。
64=2^6，因此每个Base64单元占用6个bit位，对应某个可打印字符。在计算机中数据一般按字节来来组织，每个字节占用8bit位。
在Base64编码过程中会用3个字节即24个bit位来作为处理单元，3个字节会转换为4个Base64字符。
